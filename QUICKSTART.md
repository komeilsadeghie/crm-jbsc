# راهنمای شروع سریع

## نصب و راه‌اندازی در 5 دقیقه

### 1. نصب وابستگی‌ها

```bash
# نصب وابستگی‌های بک‌اند
npm install

# نصب وابستگی‌های فرانت‌اند
cd client
npm install
cd ..
```

### 2. تنظیمات اولیه

فایل `.env` را ایجاد کنید (یا از `.env.example` کپی کنید):

```env
PORT=3001
JWT_SECRET=your-secret-key-here
```

### 3. راه‌اندازی دیتابیس

```bash
npm run init:db
```

این دستور:
- دیتابیس SQLite را ایجاد می‌کند
- جداول را می‌سازد
- یک کاربر admin با مشخصات زیر ایجاد می‌کند:
  - نام کاربری: `admin`
  - رمز عبور: `admin123`

### 4. اجرای برنامه

```bash
npm run dev
```

این دستور هم بک‌اند (پورت 3001) و هم فرانت‌اند (پورت 3000) را اجرا می‌کند.

### 5. ورود به سیستم

1. مرورگر را باز کنید و به `http://localhost:3000` بروید
2. با مشخصات زیر وارد شوید:
   - نام کاربری: `admin`
   - رمز عبور: `admin123`

## استفاده اولیه

### افزودن مشتری

1. از منوی سمت راست، "مشتریان" را انتخاب کنید
2. روی دکمه "افزودن مشتری" کلیک کنید
3. اطلاعات مشتری را وارد کنید
4. نوع مشتری را انتخاب کنید (صادرات، واردات، کوچینگ، و...)
5. ذخیره کنید

### ثبت تعامل

1. روی یک مشتری کلیک کنید
2. در صفحه جزئیات، روی "افزودن تعامل" کلیک کنید
3. نوع تعامل را انتخاب کنید (تماس، واریزی، خدمات، و...)
4. اطلاعات را وارد کنید:
   - برای واریزی: مبلغ، تاریخ، مرحله
   - برای خدمات: مدل سایت، طراح، خدمات
5. ذخیره کنید

### ایجاد هدف کوچینگ (KPI/OKR)

1. از منوی سمت راست، "کوچینگ" را انتخاب کنید
2. تب "اهداف (KPI/OKR)" را باز کنید
3. روی "افزودن هدف" کلیک کنید
4. نوع هدف را انتخاب کنید (KPI یا OKR)
5. مقدار هدف و فعلی را وارد کنید
6. مهلت را تعیین کنید
7. ذخیره کنید

### مشاهده داشبورد

1. از منوی سمت راست، "داشبورد" را انتخاب کنید
2. KPI‌های کلیدی را مشاهده کنید
3. مشتریان برتر و آخرین تعاملات را ببینید

## نکات مهم

- **امنیت**: در محیط production، حتماً `JWT_SECRET` را تغییر دهید
- **دیتابیس**: فایل دیتابیس در `database/crm.db` ذخیره می‌شود
- **بکاپ**: به صورت منظم از فایل دیتابیس بکاپ بگیرید
- **کاربر جدید**: می‌توانید از API `/api/auth/register` کاربر جدید ایجاد کنید

## عیب‌یابی

### خطای "Database not found"
```bash
npm run init:db
```

### خطای "Port already in use"
پورت 3000 یا 3001 در حال استفاده است. در فایل `.env` پورت را تغییر دهید.

### خطای "Module not found"
```bash
npm install
cd client && npm install
```
